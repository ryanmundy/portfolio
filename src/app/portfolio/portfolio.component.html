Spot the Squatch
Angular - Node - PostgreSQL
This is a simple web application built to provide those who enjoy keeping an eye out for the elusive bigfoot (or sasquatch) a way to report their own and follow and comment on other's sightings. This includes an Angular front-end that utilizes Google Maps and a Node backend which handles geocoding, reverse geocoding, and other features. This application also includes Passport authentication in addition to an admininstrator approval process for all sightings before being published.
